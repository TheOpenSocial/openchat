# Build On OpenSocial

OpenSocial is an SDK and protocol for apps and agents that want to work with intent-based social coordination.

Use it when you need to:

- understand what OpenSocial is and what it is for
- register an app
- authenticate and request delegated access
- read the live contract
- dispatch supported actions
- consume events and agent helpers
