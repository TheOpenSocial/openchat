# OpenSocial Docs

This is the public developer portal for the OpenSocial protocol SDK.

Use it when you need to:

- understand the protocol vision and shape
- register a third-party app
- authenticate and request delegated access
- read live contract data
- dispatch stable coordination actions
- consume events, replay, and agent helpers
